# **App Name**: Digital Mental Wellness: Tools, Goals, and Self-Care

## Core Features:

- Dashboard View: Dashboard with prioritized tasks, mood tracker, and recent journal entries for a quick overview.
- Task Management: Task planner with the ability to add, edit, complete and delete tasks and subtasks.
- AI Subtask Suggestions: AI-powered tool that provides subtask suggestions that helps break down tasks into actionable steps.
- Journaling: Emotional reflection and journaling to log mood, write journal entries, and track trends.
- AI Journal Insights: AI powered tool that can provide insights based on journal entries.
- Calendar View: Calendar view for tasks, displaying task counts per day and allowing to view tasks for a selected day.
- Rewards & Progress: Personalized rewards and progress section, displaying wellness points and unlocked achievements.

## Style Guidelines:

- Primary color: Deep indigo (#4F3996) to promote feelings of serenity and balance.
- Background color: Light lavender (#F5F3FF) for a soft, calming backdrop that is easy on the eyes.
- Accent color: Dusty rose (#D8B4A0) to add warmth and a touch of gentle optimism, contrasting subtly with the primary color.
- Body font: 'Inter', sans-serif, for its modern, neutral, and highly readable design, ideal for a user-friendly wellness app.
- Headline font: 'Space Grotesk', sans-serif, for a techy feel.
- Use clean, minimalist icons. Line-based icons will suit the calm aesthetic, indicating features and actions intuitively.
- Maintain a clean and spacious layout with ample white space to avoid overwhelming the user. Use a consistent grid system to align elements and provide a structured flow through the app. Cards should have soft, rounded corners to match the friendly aesthetic.
